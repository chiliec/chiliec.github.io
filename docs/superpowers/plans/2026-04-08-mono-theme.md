# Mono Theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the `landscape` Hexo theme with a new minimal `mono` theme — brutalist monospace chrome, serif body, light + dark with toggle, RSS, structured `/about/` page.

**Architecture:** New theme directory at `themes/mono/` with EJS layouts and Stylus stylesheets, both already supported by existing Hexo dependencies. The `landscape` theme stays in the repo as a reversibility net (`theme: landscape` in `_config.yml` restores the old site). Each task leaves the build in a working state.

**Tech Stack:** Hexo 8, EJS (`hexo-renderer-ejs`), Stylus (`hexo-renderer-stylus`), `highlight.js` (built-in), `hexo-generator-feed` (new). No JS framework. Vanilla JS for the theme toggle (~20 lines).

**Verification approach:** There is no JS test framework in this repo and adding one is overkill for a static site theme. Each task's "test" is a shell assertion: build the site with `npx hexo clean && npx hexo generate`, then `grep` (or `test -f`) against the generated files in `public/`. Write the assertion first, run it to see it fail, implement, run it again to see it pass — same TDD rhythm, different substrate.

**Reference spec:** `docs/superpowers/specs/2026-04-08-mono-theme-design.md`

```javascript
(function () {
  var btn = document.querySelector('.theme-toggle');
  if (!btn) return;

  function current() {
    var stored = localStorage.getItem('theme');
    if (stored === 'light' || stored === 'dark') return stored;
    return matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
  }

  function apply(mode) {
    document.documentElement.setAttribute('data-theme', mode);
    btn.setAttribute('aria-pressed', mode === 'dark');
  }

  apply(current());

  btn.addEventListener('click', function () {
    var next = current() === 'dark' ? 'light' : 'dark';
    localStorage.setItem('theme', next);
    apply(next);
  });
})();
```
